Image-Search API

  • Use the url to search for images on the web
  • Results come in the form of JSON
    • Results include:
    • url to image source
    • snippet desciption
    • thumbnail image
    • context of image location
  • url path includes api/imagesearch/'searchquery'?offset='number'&page='number'
    • searchquery: Term used for image search
    • offset: Provide the number of search results you would like returned (1-10)
    • page: Get a page of 10 new results by incrementing the page number
  • Use url/api/latest/imagesearch to see most recent search queries

Example usage:

https://obscure-springs-63954.herokuapp.com/api/imagesearch/memes?offset=10
https://obscure-springs-63954.herokuapp.com/api/imagesearch/memes?offset=10&page=2

https://obscure-springs-63954.herokuapp.com/api/latest/imagesearch
